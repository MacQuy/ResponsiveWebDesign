Overview

  • Card Name: Borrelsword Dragon

  • Card Type: Link Monster

  • Attribute: DARK

  • Type: Dragon / Link / Effect

  • Link Rating: 4

  • ATK: 3000

  • Link Arrows: Bottom, Left, Right, Top

Summoning Requirements

Materials: 3+ Effect Monsters

Card Effect

1st Effect – Attack Position Change (Once per turn):

You can target 1 Attack Position monster on the field; change it to Defense Position, and if you do, this card can make a second attack during each Battle Phase this turn.

2nd Effect – Cannot be destroyed by battle:

This card cannot be destroyed by battle with a monster in the same column.

3rd Effect – Attack Boost (Once per turn, Quick Effect):

When this card declares an attack on a face-up monster: You can activate this effect;
  • Until the end of this turn, this card gains ATK equal to half that monster's current ATK.
  • That monster’s ATK is halved.
Strategic Usage

Advantages

  • Enables OTK (One-Turn Kill) strategies due to multiple attacks and ATK boost.

  • Can break boards by halving opponent monster’s ATK.

  • Non-destruction battle protection if strategically placed.

Best Used In

  • Dragon Link, Rokket, and Code Talker decks.

  • Any deck with access to multiple effect monsters for fast Link climbing.

Rulings and Clarifications

  • Multiple Attacks: Only applies if the first effect resolves — targeting and changing a monster to Defense Position.

  • ATK Modification: The halving and gaining ATK is until the end of turn, not permanent.

  • Battle Destruction Immunity: Only applies if the monster attacking or being attacked is in the same column.

Sample Use Case

Scenario:

You control 3 Effect Monsters. Link Summon Borrelsword Dragon. Target an opponent's monster in Attack Position, switch it to Defense. Attack a monster, activate its effect to halve ATK and gain the boost. Attack again due to the first effect.

FAQs

  • Q: Can Borrelsword attack directly twice?

    A: No, the second attack must be on a monster after using the first effect.

  • Q: Does the ATK gain stack if multiple effects are used?

    A: No, the ATK gain can only be activated once per turn.

  • Q: Does Borrelsword target with its attack boost effect?

    A: No, it does not target when it activates the ATK gain effect.
